# Pensamiento-computacional.s3

## Sobre este repositorio
Esta bitacora tiene el proposito de mostrar mi proceso en el examen para pensamiento computacional :)
## Información del proyecto
### Bubble Pop
[P5.js]([https://editor.p5js.org/Tusso/sketches/pjg0-j0mW](https://editor.p5js.org/Tusso/sketches/M2_NMPC-b))

### Por Antonella Tusso
### Este código crea un juego inspirado en la solemne N°2, utilizando círculos, vibración aleatoria, mezcla de colores y crecimiento de formas para generar una experiencia interactiva y dinámica.

## Proceso
- La idea principal que tuve al ver las instrucciones del examen, fue hacer un juego de terror psicologico, al saber como la transparencia y asi queria hacer un personaje que tuviera una linterna y este pasara por distintas habitaciones que fueran incomodas para el usuario tanto por los sonidos y visualmente, lametablemente no tuve mucho tiempo para desarrollar la idea porque requeria de diseñar el personaje y las habitaciones, es una buena idea pero muy compleja.
- La segunda idea era hacer un AeroFighters pero al ver que podiamos inspirarnos de nuestra anterior solemne, quise hacer algo mas simple y bonito.
- Ya para esta idea tuve que sacar algunos codigos que no queria ocupar, como el cambio de circulo a rectangulo con la distancia del mouse, entonces solo ocupe los circulos, vibracion y mezcla de color, y agregue un codigo nuevo para que explotaran y tuvieran un cooldown.
